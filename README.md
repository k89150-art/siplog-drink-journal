# SipLog 喝飲誌

一款用來記錄飲料店、飲品客製、個人評分與回購意願的現代化前端。資料會保存在瀏覽器 `localStorage`。

## 開始使用

```bash
npm install
npm run dev
```

## 功能

- 搜尋並選擇飲料店
- 依店家搜尋飲料
- 設定甜度、冰量與最多三種配料
- 星級評分、備註與回購意願
- 最近品飲紀錄與摘要統計

設計參考稿位於 `design/siplog-dashboard-concept.png`。
